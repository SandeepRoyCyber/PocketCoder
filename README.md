# PocketCoder ⚡

> Built for you. Not built for everyone.

Turn any idea into a working personal app — instantly, offline, on your phone.

## Vision
Prompt → Local AI (Gemma 4 E2B) → JSON → WebView → Working App

## Status
- [x] Core inference engine (Gemma 4 E2B via Ollama)
- [x] Universal JSON schema
- [x] HTML renderer
- [x] Full pipeline tested — prompt to working app
- [ ] React Native Home screen
- [ ] WebView integration
- [ ] EAS Build for iPhone 17
- [ ] TestFlight distribution

## Tech Stack
React Native Expo | Gemma 4 E2B | llama.rn | WebView | Zustand | Firebase
